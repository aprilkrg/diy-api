model ideas:
- picture post( instagram) => comment
- thread (creates a forum) => comment

a thread is created with this data:
- id of post (created by db)
- user who created the post (username for 1:M, replace with id for N:M)
- title of post
- text content of post
- parent forum if applicable (forum name for 1:M, replace with id for N:M)

thread = {
    id: number,
    title: string,
    content: string,
    parent_forum: string,
    user: string
}
